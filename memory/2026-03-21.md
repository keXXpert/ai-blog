# Memory - 2026-03-21

## AI Model Testing Experiments

### Ollama Cloud Service Testing
- Tested Ollama cloud service with free tier access
- Available models: GLM-5, Minimax 2.7 (latest), QuenCode and other top-tier open-source models
- Free tier limitations quickly consumed - half of weekly limit spent
- Quality significantly better than free models available in OpenRouter
- Paid subscription costs ~$20/month and provides 50x more usage

### Minimax 2.7 Experience
- Successfully used to complete several pull requests
- Quality noticeably better than free alternatives
- However, made primitive mistakes that even junior developers wouldn't make
- Performance through Ollama free tier was adequate but left mixed impressions

### Xiaomi MIMO V2 Pro Testing
- New Xiaomi model currently available for free (likely temporary)
- Overall performance rated as "very good" by user
- Fewer reviewer comments and quick issue fixes
- Main drawback: slow performance through OpenCode (~15 minutes per feature)
- Despite slowness, completely free access makes it attractive
- Better code quality compared to Minimax 2.7

## Comparative Analysis
- Xiaomi MIMO V2 Pro: Slower but higher quality code
- Minimax 2.7: Faster but makes primitive errors
- Trade-off: Speed vs Quality balance
- OpenCode integration performance varies by model
