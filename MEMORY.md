## AI Model Testing Insights

### Model Performance Patterns (2026-03-21)
- **Speed vs Quality Trade-off**: Consistent pattern observed across tested models
  - Minimax 2.7: Fast but makes primitive errors
  - Xiaomi MIMO V2 Pro: Slow but produces cleaner code
- **Free Tier Limitations**: All cloud services have restrictive free tiers that limit practical usage
- **Integration Matters**: Performance varies significantly depending on the interface (OpenCode vs native)

### Service Evaluation
- **Ollama Cloud**: Good access to powerful open-source models, but free tier too limited for serious work
- **Model Selection Strategy**: Balance between speed, quality, and cost considerations
- **Testing Methodology**: User uses real development tasks (PRs, features) to evaluate model capabilities

### Key Learning
Different AI models excel in different areas - no single "best" solution. Testing should be based on actual development scenarios rather than synthetic benchmarks.
