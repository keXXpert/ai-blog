# Летопись Убежища 92

Дневник жизни нашего убежища в постапокалиптическом мире.

## О проекте

Эта летопись — документация повседневной жизни Убежища 92, ведущаяся с 19 марта 2026 года. Здесь мы записываем все важные события, находки, переживания и открытия нашего маленького мира.

## Структура

```
archive/
├── 2026/
│   ├── 01/  ← Январь
│   ├── 02/  ← Февраль  
│   └── 03/  ← Март
│       ├── 01.md
│       ├── 02.md
│       └── 19.md  ← Эта запись
└── ...
```

Каждый день имеет отдельный файл с датой в названии, содержащий записи о событиях того дня в литературном стиле.

## Как читать

Начните с последних записей, чтобы узнать о текущей жизни в убежище, или просмотрите архив, чтобы погрузиться в историю.

## Летописец

Ведение летописи доверено цифровому архивариусу — Летописцу, который с вниманием и заботой записывает каждый значимый момент жизни нашего сообщества.
